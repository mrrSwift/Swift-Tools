## Installation

Install my-project with npm

```bash
  npm install swiftlly-tools
```

```js
const swiftly = require('swiftly-tools')
const str = 'Hello'
swiftly.string.isLatin(str) // true
```


    
## Usage/Examples

```js
const swiftly = require('swiftly-tools')

const str = 'Hello'
swiftly.string.isLatin(str) // true

app.use(swiftly.middleware.cors)
```
